SCSS for Active Tabs

I’ve seen a couple of different ways for an active tab to get rid of the bottom border. Either a `margin-bottom: -1px;` or a `top: 1px;`, but here is a great method that doesn’t alter the layout or height of your tabs:

 
.active {
  position: relative;
  background-color: $menu-tabular-item-active-background-color;
 
  &:after {
      display: block;
      content: "";
      position: absolute;
      bottom: -1px;
      width: 100%;
      left: 0;
      border-bottom: 1px solid $menu-tabular-item-active-background-color;
  }
}

Here the :after class is used to hide the bottom border. Works in all modern browsers and IE 9+.
